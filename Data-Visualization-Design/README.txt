# Responsive Dashboard with Light & Dark Mode

## Description
This project features the creation of a **responsive dashboard** with support for **Light** and **Dark Mode**. Using HTML, CSS, and JavaScript, the dashboard presents statistics like "Users", "Sessions", "Revenue", and "Growth", along with interactive charts for **User Distribution** and **Monthly Revenue**.

The key feature of the dashboard is the ability to toggle between Light and Dark Mode with a **toggle switch**, while the charts are generated using the **Chart.js** library.

## Features
- **Light & Dark Mode**: Toggle between themes with a switch.
- **Responsive Design**: Dashboard adapts to all devices.
- **Card Components**: Displays statistics such as Users, Sessions, Revenue, and Growth.
- **Charts**: Pie and bar charts are used to display data (User Distribution, Monthly Revenue).
- **Interactive UI**: Toggle between themes and dynamically present data via charts.
